Tabuadas de 1 a 10

Foi usado o codigo abaixo para realizar o exercício, veja o resultado no console do inspecionar

            for (let i = 1; i <= 10; i++) {
                console.log(`Tabuada de ${i}`);
                for (let j = 1; j <= 10; j++) {
                    const result = i * j;
                    console.log(`${i} x ${j} = ${result}`);
                }
                console.log('');
            }
